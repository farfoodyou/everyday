#### 为什么要用这个方式来记录
突然发现用atom来管理文档还是非常方便的。
然后markdown来记录在安装配置软件的过程中，截图并且插入别比较方便。
而且它可以内嵌HTML代码来完成对格式的要求。一些事，例如：     
<font color = red size = 3px>红色</font>
<font color = yellow size = 3px>黄色</font>   
虽然在github上好像这种方式是无效的。

还有编写说明文档等。 
